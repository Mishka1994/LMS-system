# Система управления обучением (LMS-system)

Проект позволяет пользователям покупать и подписываться на разные курсы и уроки, а также создавать собственные. Реализована система подписок на курсы через оплату по API платформы Stripe.
Реализована система уведомлений через Telegram об изменениях на курсе.

## Для установки и запуска проекта
1. Клонировать проект в локальный репозиторий: `git clone https://github.com/Mishka1994/LMS-system.git`
2. Создать виртуальное окружение в корневой папке проекта: `python -m venv .venv`
3. Установить зависимости для проекта: `pip install -r requirements.txt`
4. Запустить проект: `docker-compose up --build`

Адрес для доступа проекта ***http://localhost:8001***
